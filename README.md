🪄 Mana War (가명) (마나 워)
AI 보조를 통해 개발된 웹 기반 실시간 전략/액션 게임 프로젝트입니다. 
게임을 통해 주식 용어와 차트에 대해서 쉽고 간편하게 배울 수 있는 프로젝트입니다.
간단한 웹 브라우저 환경에서 마나(자본금)를 관리하며 전투를 즐길 수 있도록 설계되었습니다.
추후에 어플로도 제작을 진행할 예정입니다.

🚀 프로젝트 개요
설명: AI 기술을 활용하여 코드 구조를 설계하고 구현한 웹 게임입니다.

📖 프로젝트 스토리
		
유저는 시장의 불합리함 속에서 엄청난 피해를 입은 후 자신과 같은 처지의 사람들과 모여 
자신들에게 피해를 준 시장에게 복수를 하러 떠나는 모험

📂 파일 구조
index.html: 게임의 기본 레이아웃 및 UI 구조

style.css: 게임 화면의 디자인 및 애니메이션 처리

app.js: 게임 로직, 캐릭터 컨트롤 및 전투 시스템 핵심 코드

mana_war_update_v6.patch.diff: 최신 업데이트 내역 및 패치 로그

🛠 주요 특징 (예시)
AI 기반 코드 최적화: 개발 과정에서 AI를 적극 활용하여 효율적인 로직 구현

반응형 인터페이스: 웹 환경에서 직관적으로 조작 가능한 UI
